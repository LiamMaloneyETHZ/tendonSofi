THIS IS THE PERSONAL AND FISH VERSION OF THIS REPO

1. "DynamixelSDK" holds the main default dynamixel library

2. "dxlSetup" holds the modified servo class definitions and functions, port handling, and group syncs

3. "uwSnakeFiles" holds files related to the underwater limbless robot, but demonstrates helpful implementations of the library and unique functions to different applications such as "tlsFuncs2" (extra functions) and hotkeyControl (example top-level script)

4. "rotatingJointFiles" holds files related to the rotating joint control

5. "prev" holds unused stuff
